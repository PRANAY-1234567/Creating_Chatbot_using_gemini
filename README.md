🤖 Gemini Chatbot Desktop App (Python + CustomTkinter)

A modern desktop chatbot application built using Python, CustomTkinter, and Google Gemini API.
The app provides a clean, advanced UI for chatting with Google’s Gemini 1.5 Flash model in real time.

🚀 Features

Modern GUI using CustomTkinter

Sidebar-based layout (like professional chat apps)

Scrollable chat area with message bubbles

Real-time responses from Google Gemini API

Keyboard support (Enter to send)

Clear chat functionality

Responsive window resizing

🛠️ Tech Stack

Python 3


CustomTkinter

Requests

Google Gemini API (Generative Language API)

📦 Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/gemini-chatbot-gui.git
cd gemini-chatbot-gui

2️⃣ Install required libraries
pip install customtkinter requests

🔐 API Key Setup (Important)

Go to Google AI Studio

Create an API key for Gemini

Replace the API key in the code:

GEMINI_API_KEY = "YOUR_API_KEY_HERE"

⚠️ Security Warning

Do NOT upload your real API key to GitHub.

Add this to .gitignore or use environment variables:

.env

▶️ How to Run
python gemini_chatbot.py

🧠 How It Works

User input is sent to the Gemini API

API responds with generated text

Messages are displayed as chat bubbles

UI updates dynamically with scrolling

Sidebar allows clearing the chat
